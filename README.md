# devroast

O **devroast** e um app para colar trechos de codigo e receber uma analise com humor, sarcasmo e um toque de caos controlado.

## O que o app faz
- Analisa snippets de codigo com feedback em linguagem divertida.
- Mostra score de "qualidade" de forma visual.
- Destaca pontos criticos com badges, cards e diff lines.
- Exibe um preview de ranking estilo **hall of shame**.
- Traz uma interface dark inspirada em terminal/editor.

## Estado atual
- Projeto em evolucao com dados mockados (sem integracao com API por enquanto).
- Componentes visuais reutilizaveis e composaveis para acelerar novas telas.

## Contexto
Este app esta sendo construido durante o evento **NLW** da **Rocketseat**, acompanhando as aulas e evoluindo por iteracoes ao longo do evento.

## Como rodar
```bash
pnpm install
pnpm dev
```

Abra `http://localhost:3000` no navegador.
